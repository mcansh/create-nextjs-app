# create-nextjs-app

[![Greenkeeper badge](https://badges.greenkeeper.io/mcansh/create-nextjs-app.svg)](https://greenkeeper.io/)

Quickly start a new nextjs project

## Installation
```bash
  yarn global add create-nextjs-app
```

## Usage

```bash
  mkdir my-new-app && cd my-new-app
  create-nextjs-app
```
